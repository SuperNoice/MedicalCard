Ключи для автозаполнения документов:

{fio} - ФИО Пациента
{sex} - Пол Пациента
{birthDay} - Дата рождения Пациента
{address} - Адрес Пациента
{phone} - Телефон Пациента
{countryType} - Тип места проживания Пациента
{passport} - Пасспортные данные Пациента (Полностью)
{nowd} -  День текущего времени
{nowM} - Месяц текущего времени (цифра)
{nowMMMM} - Месяц текущего времени (Слово полностью)
{nowyear} - Год текущего времени (4 цифры)


*Внимание! Некоторые закладки(Bookmarks) документа могут вызывать ошибку печати. Старайтесь избегать их использования.
